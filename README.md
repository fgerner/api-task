# api-test-task

Prerequisites
  Node
  Mocha

#Install

Clone this repo
`git clone https://github.com/fgerner/api-task.git`

Change to project directory
`cd api-task`

Install 

`npm install`

#Run Tests
- Preview site
`npm run test:preview`

- Demo site //Needs configuration uppdate, enpoint wasn't responding. And will require SSL certificate
`npm run test:demo`
